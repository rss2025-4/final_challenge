The top left pixel is the origin, u axis increases to right, and v axis increases down.

“xy” is of the ground, in meters

final_challenge.homography.Point
A parameter annotated with Point can take one of

  1. (x, y)

  2. (x, y, z), in projective coordinates. this is equivalent to the euclidean point (x/z, y/z).

These can be python or numpy.

functions returning Point mostly returns in project coordinates. use point_coord() to convert to euclidean

alias of ndarray | tuple[ndarray | float, ndarray | float]

final_challenge.homography.Line

(a, b, c), represents the line ax + by + c = 0

can be python or numpy.

construct a Line with line_from_slope_intersect(), line_through_points()

The “positive side” is points (x, y) such that ax + by + c >= 0

The “direction” of the line is, direction == up <=> left is “positive side”

alias of ndarray | tuple[ndarray | float, ndarray | float, ndarray | float]
